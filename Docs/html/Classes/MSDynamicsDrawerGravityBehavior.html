Inherits from MSPanePositioningBehavior : UIDynamicBehavior
Declared in MSDynamicsDrawerBehavior.h
MSDynamicsDrawerBehavior.m

Overview

Uses gravity to move the pane between states. When the pane arrives at its new state, it bounces against a boundary until it reaches a resting position.

Internally, MSDynamicsDrawerGravityBehavior uses UIGravityBehavior and UICollisionBehavior child behaviors to position the pane.

Properties

gravity

The child behavior that is used to create the gravity effect on the pane.

@property (nonatomic, strong, readonly) UIGravityBehavior *gravity

Discussion

Magnitude is the only property that should be modified on this behavior to change the positioning effect when using this behavior. This behavior’s behavior is undefined if other properties are modified.

positionPaneInState:forDirection:

Positions the pane dynamic item in the desired direction with the specified state. This method is only invoked internally by the associated MSDynamicsDrawerViewController. It should not be invoked otherwise. When subclassing MSPanePositioningBehavior, this method should be overridden to adjust child behaviors to create the desired effect.

- (void)positionPaneInState:(MSDynamicsDrawerPaneState)paneState forDirection:(MSDynamicsDrawerDirection)direction

Parameters

paneState

The state that the pane should be positioned in. Represented as targetPaneState after this method is invoked and until this behavior is removed from its dynamic animator.

direction

The direction that the pane should be positioned in. Represented as targetDirection after this method is invoked and until this behavior is removed from its dynamic animator.
